# agente_IA_atualizacao_ppt
Desenvolvi um agente de IA que permite aos analistas solicitar, em linguagem natural, a atualização automática de apresentações em PowerPoint. O usuário informa parâmetros como cidade, ano de análise, período de evolução, intervalos de CAGR e microáreas de análise, e o agente processa os comandos para gerar o relatório atualizado.
